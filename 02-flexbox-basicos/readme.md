Default properties are set when the parent container's display is: `display: flex;`

## To container configuration

`flex-direction: row;`

`justify-content: flex-start;`

`align-items: stretch;`


## To items configuration
`flex-grow: 0;`

`flex-shrink: 1;`

`flex-basis: 0%`

## Concepts
`flex-wrap` (flex-container)
* allows you to group content

`flex-basis` (flex-items)
* initial value of an item
* supports relative or absolute widths

`flex-grow` (flex-items)
* element growth factor

`flex-shrink` (flex-items)
* element shrink factor

`flex shorthand` (flex-items)
* `flex-grow | flex-shrink | flex-basis`
